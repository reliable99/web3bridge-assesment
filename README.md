# web3bridge-assesmentSavings-group-html-js
README.md
Savings Group Web App (HTML, CSS, JavaScript)

A visually appealing web-based savings group application simulating a student collective investment in a Play-to-Earn blockchain game. This project uses HTML, JavaScript, and external CSS with a modern, clean, and unique design.

Features

Student Registration

Savings Dashboard

Weekly Simulation

Membership Management

Tier Information
Tier	Amount (Naira)	Interest per Week
Tier 1	10,000	5%
Tier 2	20,000	10%
Tier 3	30,000	20%
How to Use

Open index.html in a browser. Fill the form, select a tier, simulate weekly interest, and manage withdrawals.

Files

index.html - HTML layout

style.css - Modern, beautiful external CSS

app.js - JavaScript logic
